# P0
**组织的核心可信业务数据资产**
- 用于核心指标与数据看板，支撑业务及产品决策
- 用于向监管机构、政府部门及第三方开展关键合规报备
- 用于对品牌或营收存在重大影响的线上用户交互场景（搜索结果、广告投放、促销活动及实验测试）
- 其他高价值应用场景，如机器学习模型与欺诈检测
- 用于衍生其他核心一级数据集的数据源

# P1
**公司的重要业务数据集（重要性低于一级数据集）**
- 用于重要业务指标、产品指标及数据看板，支撑内部决策
- 用于向主要监管机构、政府部门及第三方提交重要合规报告
- 用于重要性较低的线上用户交互场景（用户活跃度、用户行为）
- 用于衍生其他关键二级数据集的数据源

# P2
**部门 / 团队级数据集，通常为非业务类通用内部系统数据**
- 用于产品指标及数据看板，支撑产品决策
- 用于跟踪内部系统的运营指标
- 用于衍生其他关键三级数据集的数据源

# P3
**团队级数据集，通常为非业务类通用内部系统数据**
- 用于产品指标及数据看板，支撑团队决策
- 用于跟踪该团队所属内部系统的运营指标
- 作为数据源，用于生成其他关键的四级数据集

# P4
**私有 / 闲置数据资产—— 仅对个人用户产生影响，无其他外部影响**
- 无归属主体、且过去 60 天内未被使用的数据资产
- 归个人所有、无团队归属的数据资产
