## 方案概要（开发类）

技术方案概述。包括：

- 分层归属（新表在哪个分层 / 修改哪些已有表）
- 数据来源（上游依赖哪些表）
- 关键设计决策（粒度、分区策略、分桶键）
- 特殊处理（NULL 兜底、脏数据过滤、口径说明）

## 方案概要（分析类）

分析方法概述。包括：

- 分析范围（哪些 schema / 业务域 / 对象类型）
- 评估框架（使用什么规则/标准，引用哪个 SHARED/knowledge 文件）
- 关键设计决策（分析粒度、分批策略、边界处理方式）
- 数据采集方案（需要哪些数据、如何获取）
